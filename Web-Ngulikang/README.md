# NguliKang - Platform Jasa Konstruksi & Tukang Profesional

Platform digital yang menghubungkan pemilik proyek dengan tukang dan pekerja konstruksi berpengalaman di Indonesia.

## Tentang Project

NguliKang adalah solusi modern untuk kebutuhan konstruksi dan renovasi rumah. Kami menyediakan akses mudah ke tukang profesional yang sudah terverifikasi, sistem pembayaran yang aman, dan tracking progress real-time untuk setiap proyek.

Project ini dikembangkan menggunakan React + Vite untuk performa optimal dan developer experience yang lebih baik.

## Fitur Utama

- **Cari Tukang Profesional** - Filter berdasarkan keahlian, lokasi, dan rating
- **Sistem Booking Online** - Pesan jasa tukang langsung dari platform
- **Tracking Proyek** - Monitor progress pekerjaan secara real-time
- **Pembayaran Aman** - Sistem escrow untuk keamanan transaksi
- **Review & Rating** - Sistem feedback dari pengguna sebelumnya
- **Live Chat** - Komunikasi langsung dengan tukang

## Tech Stack

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Styling**: CSS-in-JS (inline styles)
- **Animations**: Framer Motion
- **Routing**: React Router DOM
- **Icons & Assets**: Custom SVG + local images

## Getting Started

### Prerequisites

Pastikan sudah install:
- Node.js (v16 atau lebih baru)
- npm atau yarn

### Installation

1. Clone repository
```bash
git clone https://github.com/yourusername/nguli-react.git
cd nguli-react
```

2. Install dependencies
```bash
npm install
```

3. Jalankan development server
```bash
npm run dev
```

4. Buka browser di `http://localhost:5173`

## Available Scripts

- `npm run dev` - Jalankan development server
- `npm run build` - Build untuk production
- `npm run preview` - Preview production build
- `npm run lint` - Check code quality
